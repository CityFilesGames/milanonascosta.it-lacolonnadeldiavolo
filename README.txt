Per pubblicarla su GitHub Pages:
1. crea un repository
2. carica tutti i file di questa cartella nella root del repository
3. vai su Settings > Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. salva

Dopo la pubblicazione, la pagina userà il video locale in autoplay, muted, loop e playsinline.
