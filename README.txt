MILANO NASCOSTA - FILE PRONTI PER GITHUB PAGES

Contenuto cartella:
- index.html
- styles.css
- video.mp4

Pubblicazione rapida su GitHub Pages:
1. Crea un nuovo repository su GitHub
2. Carica questi 3 file nella root del repository
3. Vai in Settings > Pages
4. In Build and deployment seleziona:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
5. Salva
6. Attendi il link pubblico del sito

Nota:
- Su mobile l'autoplay con audio è quasi sempre bloccato dai browser.
- Questa versione prova ad avviare il video automaticamente in muto.
- Se il browser non lo consente, compare sempre il pulsante "Avvia video".
